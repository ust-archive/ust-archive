## Disclaimer

This repository indexes data aggregated and/or scraped from HKUST (the University) web resources. By using the datasets indexed by this project, you acknowledge the following:

- **No Official Affiliation**: This project is not maintained, monitored, or supported by HKUST. It is a community-led initiative. The data provided here is for informational purposes and does not constitute an official University record.

- **Data Integrity**: All data is provided "as-is." While the project strives for accuracy, the official University website remains the sole authoritative source of truth. The maintainers are not responsible for discrepancies between this index and official records.

- **Visibility Policy**: This project intends to follow the same visibility level as the original data source. Public data is indexed publicly; private data remains indexed privately.

If you represent HKUST and have concerns regarding the indexing of this data, please open an Issue and we will address it promptly.
